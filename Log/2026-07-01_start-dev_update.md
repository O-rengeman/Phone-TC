# 起動スクリプトの修正

## 実施日
2026-07-01

## 目的
ローカル開発サーバー起動用の `start-dev.bat` について、アプリケーション名の変更を反映し、スマートフォン端末からP2Pテストなどが行えるように修正を行うこと。

## 修正内容
- **文字化け防止**: Windows コマンドプロンプトでの表示文字化けを防ぐため、`chcp 65001 > nul` を追加し UTF-8 に固定しました。
- **タイトルと出力メッセージの変更**: `Phone-TC` と表記されていた部分を、正しい名称である `LTC SYNC PRO` に修正しました。
- **外部アクセス許可**: ローカルネットワーク内のスマートフォン等から検証可能にするため、Vite の起動コマンドに `--host` フラグを追加（`npm run dev -- --host`）しました。

これにより、起動用バッチファイルでも「LTC SYNC PRO」として正しく機能し、実機テストがスムーズに行えるようになりました。
