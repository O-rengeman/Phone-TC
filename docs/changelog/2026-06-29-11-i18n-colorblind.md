# ✨ 日本語化(i18n) と 色覚配慮(色＋文字/形)

- **日付**: 2026-06-29
- **種別**: 機能追加（アクセシビリティ / 国際化）
- **主な対象ファイル**: `src/utils/i18n.ts`(新規), `src/utils/i18n.test.ts`(新規), `src/App.tsx`, `src/App.css`

## 背景

JPユーザー/市場に向けた日本語対応と、屋外の照り返しや色覚特性でも状態を判別できるUI（色だけに依存しない）を実現する。

## 変更内容

### 日本語化 (i18n)
- 新規 `i18n.ts`：依存ゼロの軽量翻訳層。`t(key, lang, vars?)` でEN/JA辞書を引き、未定義キーは英語→キー名にフォールバック、`{var}` 補間対応。言語は localStorage→ブラウザ言語の順に判定（既定EN）。純ロジックでテスト6件。
- ヘッダーに **言語トグル（日本語/EN）** を追加。選択は localStorage に保存。
- タブ／状態ピル／各セクションラベル／メインボタン（開始・停止・長押しで停止・準備中・再開）／ドリフト精度パネル／MAIN同期バー／接続ガイド全文／マーカー関連／スレート文言、および主要トーストを翻訳キー化。
- 技術用語（FPS値, MIC/LINE, STEREO TC, L-TC/R-AUDIO, EDL/ALE, HEX 等）は誤解防止のため原文のまま。

### 色覚配慮（色＋文字/形）
- マーカーのスウォッチに**色名イニシャル（R/B/G/Y）**を表示（フッターの打点ボタン、ログ行のドット、確認フラッシュ）。色相に依存せず識別可能。
- 状態ピルのドットを**形でも区別**（待機＝中抜きの輪郭、準備中＝点滅、出力中＝塗り＋発光）。状態テキスト（待機/同期中/LTC出力中）と併せ二重符号化。
- クライアントのドリフト表示に **✓ / ⚠ 記号**を付与（色のみ依存を解消）。
- 精度バッジ（HIGH/MEDIUM/LOW）や同期モードは元々テキスト併記のため変更なし。

## 影響・検証

- `tsc -b` 通過、ESLint クリーン、テスト **86件合格**（i18n 6件追加）、Viteビルド **64モジュール成功**。
- ロジックの中核は不変。トースト翻訳は `langRef` 経由で参照し、useCallbackの依存に影響しない設計。

## 補足

辞書は EN/JA フル対訳。新規文言は `i18n.ts` の `en`/`ja` に1行追加すれば反映される。さらなる言語追加も `DICTS` にロケールを足すだけで対応可能。
